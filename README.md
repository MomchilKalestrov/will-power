## Building

### Installing dependencies

`npm i`

### Starting dev server

`npm run dev`

### Start production server

`npm run build`

`npm run start`

### Env vars

To run properly add the `MONGODB_URI`, `BLOB_READ_WRITE_TOKEN`, `NEXTAUTH_URL` and `NEXTAUTH_SECRET` environment variables.
